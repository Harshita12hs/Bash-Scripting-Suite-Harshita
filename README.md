## Overview
This project demonstrates the use of Bash scripting to automate essential system operations.  
It includes a collection of scripts that perform **backup, cleanup, monitoring, and maintenance** tasks on a Linux-based environment.

---

## Project Structure

| Script Name | Description |
|--------------|-------------|
| `backup_script.sh` | Automates creation of backups for project files |
| `update_cleanup.sh` | Simulates system update and cleanup process |
| `log_monitor.sh` | Monitors log files and alerts if errors or warnings are found |
| `maintenance_suite.sh` | Menu-driven master script to run all utilities interactively |
| `myscript.sh` | First test script to verify Bash setup |
| `suite_log.txt` | Central log file storing all script activities |

---

##  Features
- Automated file backup with timestamps  
- Simulated system cleanup and update  
- Real-time log monitoring for errors and warnings  
- Menu-driven control center for easy script execution  
- Centralized logging with error handling
